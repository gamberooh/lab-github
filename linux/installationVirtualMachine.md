# Installazione Linux su Macchina Virtuale
## 1) Scarica l'immagine disco ISO di Linux Mint
Ultima versione: https://linuxmint.com/download.php 

## 2) Scarica il tuo gestore di Macchine Virtuali
VirtualBox è un gestore di macchine virtuali a livello applicazione per Windows. 
UTM è un gestore di macchine virtuali a livello applicazione per MacOS.
Praticamente puoi eseguire un sistema operativo linux sulla tua macchina fisica Windows o Mac!

## 3) Creo una Macchina Virtuale con le risorse che gli voglio affidare
Puoi selezionare la quantità di CPU, RAM e GPU che vuoi affidare alla tua Macchina Virtuale. (Linux non ne ha bisogno di tante)

## 4) Seleziona la ISO di Linux Mint
Vai nella cartella in cui hai scarcato la ISO e aggiungila.

## 5) Avvia la VM
Hai finito! Ma in parte... ;( Puoi avviare la VM e fare l'ultimo passo.

## 6) Fai l'installazione guidata di Linux Mint
Linux Mint ti fornisce già una installazione guidata e dovrai selezionare adeguatamente.
 - Fuso orario
 - Lingua
 - Layout Tastiera
Ora hai finito realmente! Goditi Linux <3
